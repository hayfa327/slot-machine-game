##  What It Does

This game simulates a classic slot machine:

* Click the **Spin** button to randomize the wheels
* Try your luck to get matching symbols
* Add sound and animation for feedback

It's a playful, beginner-friendly project demonstrating core JavaScript skills with visual UI.

---

##  Features

-  Realistic slot spinning simulation
-  Randomized results using JavaScript
-  Win/lose detection
-  Styled with CSS for engaging UI
